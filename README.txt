Проект „Автострада“

Игровой процесс:
Игрок может управлять машиной, основня задача — уворачиваться от других машин, движущихся навстречу игроку.
Чтобы начать игру, игрок должен нажать на кнопку „start the game“.
 Управление машиной осуществляется с помощью стрелок на клавиатуре, одномоментно игрок может двигаться только в одном направлении,также игрок не может выезжать за пределы дороги или экрана. При столкновении с другими автомобилями у игрока отнимаются жизни, которые отображаются в левом верхнем углу окна.Когда жизни игрока будут израсходованы, ему будет предложено начать игру заново, он сможет сдеоать это нажав на кнопку „New game“.
Трафик не сталкивается между собой, при угрозе столкновения машина едущая сзади снижает скорость
Реализация:
Классы программы:
Класс Hud. Отвечает за отображение здоровья игрока
Класс Player. Отвечает за машину игрока
Класс Traffic. Отвечает за остальные машины
Класс Background. Отвечает за дорогу
Класс Button — вспомогательный класс для создания кнопок
Класс EndGameImg. Отвечает за изображение, появляющееся в конце игры
